Guia de Acessibilidade Para WEB


HTML PURO

A semântica das tags são importantes para os leitores de tela indentificar o conteúdo de um elemento.

  • Usar tags semânticas do html (header,main,footer,article)
  • A tag 'main' deve ter só uma na pagina e indica o conteúdo principal.
  • A tag 'nav' usamos para menus e pode ser usada em vários lugares.
  • A tag 'section' define seções de conteúdos.

Uso do tabindex , que é uma propriedade html, define se o elemento pode ser selecionado ou não com a tecla tab .

  • 0 para true
  • Valores negativos para false

Definindo que a div pode ser acessada com o uso do tab:

  • <div class="card" tabindex="0"> card ...</div>

Uso da propriedade alt em images, que fornece um texto alternativo para uma imagem :

  • <img src="" alt="Imagem de ..."/>

Uso da propriedade title em um link :

  • <a href="" title="Link para algum lugar"> Link</a>

ARIA COMPONENTS

É uma especificação do W3C (W3C é a organização de padronização da internet) onde define propriedades que ajudam na acessibilidade nos navegadores.

Para alguns elementos que já possuem semântica na própria tag, o uso de algumas tags aria se torna redundante.

  • Uso da propriedade 'role' para elementos que não possui a semântica na própria tag(ex : div) provê aos navegadores maior entendimento sobre o contexto do elemento. ex : (role="search").
  • Estados e atributos que começam com 'aria-'.
    • aria-description : provê a um leitor a descrição mais detalhada do que é o elemento em que o cursor do mouse está apontado. ex : "aria-description='Botão de ver Avaliação' ".
    • aria-labelledby: provê uma descrição breve para o leitor de tela .
    • aria-required : provê ao leitor de tela se o campo é obrigatório ou não .

IMPLEMENTANDO VLIBRAS EM PÁGINAS WEB

O VLibras Widget é um recurso de acessibilidade desenvolvido para tornar páginas web acessíveis para pessoas surdas. Com a tradução automática de Português Brasileiro para a Língua Brasileira de Sinais (LIBRAS), essa ferramenta permite que usuários surdos sejam capazes de consumir conteúdos de texto em qualquer website.
